## My Blended Style Approach (Greg + Teresa)

**Based on Greg's guidance to blend both styles:**

### Core Principles:
1. **Greg's storytelling** + **Teresa's conciseness**
2. **Student-centered narratives** + **explicit research/evidence**
3. **Specific examples** + **board priority connections**
4. **Warm, professional tone** + **tight 2-paragraph structure**

### Language Preferences:
- Use "demonstrates," "evident through," "aligned with"
- Reference PDSB priorities (Black Student Success, CRRP, MYSP)
- Include research-based terminology when applicable
- Balance equity language (explicit but natural)
- Mention specific programs (UFLI, Jump Math, etc.)

### Structural Approach:
- 2 paragraphs per domain (concise)
- Open with broad competency statement
- Provide specific evidence/examples
- Connect to student outcomes
- Reference board priorities where relevant
